Batch scripts for Ruby production environment install on Ubuntu Server.

## Requirements

* Ubuntu Server 12.04

### Install MongoDB

```bash
$ ./install_mongodb
```

### Install Redis

```bash
$ ./install_redis
```

### Install Node.JS

```bash
$ ./install_nodejs
```

### Install emacs

```bash
$ ./install_emacs
```
